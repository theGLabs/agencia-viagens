# 🌍 Horizonte Viagens

Site de agência de viagens desenvolvido com **HTML**, **CSS** e **Bootstrap 5**.

## ✨ Funcionalidades

- Navbar responsiva com colapso no mobile
- Hero section com animação de zoom
- Filtro de destinos por categoria (Nacional / Internacional)
- Cards de destinos com hover animado
- Seção de pacotes com destaque para o mais vendido
- Seção "Sobre" com estatísticas da agência
- Formulário de contato com validação nativa HTML5
- Animações de entrada ao rolar a página
- Layout totalmente responsivo

## 🛠️ Tecnologias

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons

## 📄 Seções

- **Início** — hero com chamada para ação
- **Destinos** — cards com filtro nacional/internacional
- **Pacotes** — planos e preços
- **Sobre** — história e estatísticas da agência
- **Contato** — formulário de contato
